# 02 Reference Router And Locks

priority: high
scope: @图片 / @视频 / @音频 / 序列帧 / 参考提示词的用途分配和维度锁定

## 总原则

场景锁定、角色锁定、音频锁定，本质都是说明参考素材只控制哪些维度。默认合并写进 `【角色/场景/素材锁定】`，不要再拆成 `【参考素材使用】`、`【场景锁定】`、`【角色锁定】` 三个重复模块。不能写“参考 @图片1”就结束。

默认用户可见输出只写“控制什么”：

```text
@素材N 命名为 [锁定名]，控制 [维度列表]。
```

只有以下情况才显性补充“不控制什么”：

- 多个参考素材维度冲突。
- 参考视频可能把人物身份、服装、地点、品牌、台词或声音带偏。
- 参考音频可能被误当成最终 BGM、台词或声线。
- 涉及真实人物、声线、品牌、logo、IP、授权不明素材。
- 用户要求严格角色一致、产品一致、首尾帧一致，且参考素材中有容易误读的额外元素。

高风险时使用：

```text
@素材N 命名为 [锁定名]，控制 [维度列表]；不控制 [具体排除维度]。
```

时间轴引用规则：

- 锁定名只在**当前独立段落内部**复用。进入下一独立段后，重新写该素材的原始引用、锁定名和控制维度；不能依赖项目总说明或上一段已经命名过。
- 在 `【角色/场景/素材锁定】` 里为素材命名后，`【运镜与时间轴】` 默认使用锁定名，不频繁重复 `@图片1/@视频1/@音频1`。
- 正确：`CUT 1，主角A从screen-left进入稻田场景，收割机道具停在background right。`
- 错误：`CUT 1，@图片1人物从screen-left进入@图片2场景，@图片3道具在右侧。`
- 只有新素材第一次出现、需要强调首帧/尾帧承接、或修复参考漂移时，才在时间轴中再次写原始 @引用。

跨段重复门：

- 角色、服装、场景、关键道具、首/尾帧、声线与文字资产只要在某段实际出现，就在该段 `【角色/场景/素材锁定】` 重新声明用途和当前状态。
- 可读文字、报告、证件、UI、包装或屏幕截图使用固定贴图/跟踪合成时，每个出现它的独立段都重复锁定全部需要保持的文字、数字、单位、箭头、表格线和版式；特写只裁切或缩放原图像素，不让“整片锁定”退化为上一段的隐含设定。
- 总资产表可以服务人类管理，但不能替代可复制提示词块内的本段锁定。

## 多模态素材路由器

@图片可控制：

- character identity：脸型、发型、年龄感、体型、服装、道具。
- environment identity：地点、空间布局、建筑材料、天气、时代。
- style identity：画风、材质、色彩、构图气质。
- first frame / last frame：首帧或尾帧状态。

@视频可控制：

- camera movement：运镜路线、速度、转场方式。
- action choreography：动作路线、接触点、节奏、反击、结尾后果。
- editing rhythm：切点、镜头长短、反应镜头结构。
- VFX behavior：粒子、烟尘、能量、破碎方式。
- audio rhythm：如果用户明确要求，也可作为节奏参考。

@音频可控制：

- final BGM：最终背景音乐。
- silent beat guide：只控制节奏，不输出该音频。
- SFX guide：冲击、转场、动作音效参考。
- ambience bed：风、雨、机械、室内空间声。
- voice anchor：角色或旁白声线参考。

序列帧 / 分镜图可控制：

- 镜头顺序、空间轴线、screen-left/right、主体进出画、焦点路径、首尾状态。
- 不自动控制角色身份、服装、场景质感，除非用户明确指定。

参考提示词可控制：

- 结构、节奏、动作拆法、时间轴写法、画面密度。
- 不默认复制题材、角色、世界观、专有名词。

## 素材证据审计

给素材分配控制维度前，先静默分三层：

1. **能证明**：画面或声音中直接可见/可听的身份、服装、材质、构图、光位、初始姿态、尺度锚点、动作方向、节奏或声线。
2. **仅暗示**：画外空间、下一步动作、速度、重量、完整环境、反向机位等需要推断的内容。
3. **不能证明**：素材未显示的背面、内部结构、真实尺寸、反向轴线、动作结果、台词含义或版权归属。

规则：

- 每个控制维度只设一个主参考；其他素材只能补充不同维度，不能共同争夺同一身份、构图或动作路线。
- 场景图没有反向机位或新角度证据时，它只控制材质、光线和已见布局；新机位必须用文字重新建立轴线、左右、前中后景和镜头高度，不能强套原构图。
- 参考图中的速度线、倾斜姿态、衣料方向、运动模糊和被截断肢体属于“隐含运动”。如果它与用户要求的动作方向或镜头路线冲突，先解决冲突再使用。
- 色卡只控制色彩比例、曝光关系、冷暖和光线角色，不控制构图、文字、人物或场景布局。
- 灰底角色/产品图只控制身份、轮廓、材质、服装、道具和尺度特征，不把灰底带入场景。
- 尺度用人物身高、门高、车辆宽度、地面占地或明确米制尺寸等绝对锚点证明，不用两个都会漂移的对象互相作比例。

## 联网核实路由

只有现有知识不足且错误会改变画面执行时才静默联网：陌生/非标准动作，钢丝、威亚、特技或特殊摄影装置，罕见摄影机路线，机械/材料/工业流程，时代或文化特定行为，真实安全流程，或用户点名名场面且需要核实其可迁移镜头机制。

- 优先官方、专业和一手来源：摄影协会/电影资料机构、设备厂商文档、主创/特技/VFX 访谈、论文和技术文档。
- 只提取可执行增量：准确术语、发力与支撑、camera start-path-end、轴线、接触点、切点和安全边界。
- 常见镜头和知识库已有动作不联网，避免噪声与延迟。
- 术语不可靠时，不把术语硬塞进提示词，改用中文描述真实物理路线。
- 研究网址、来源争议和检索笔记只留在分析或制作备注中，不进入可复制的视频提示词。

## 正史参考与临时状态

长项目和续写必须区分两类素材：

- 正史参考 / canonical reference：角色脸、服装、产品形状、场景设计、道具、色彩等不变量。
- 已接受画面 / accepted footage or final frame：上一段实际生成后的姿态、位置、动作相位、镜头相位、声音相位、临时环境状态。

规则：

- 角色图、产品图、场景图控制稳定身份和设计。
- 已接受上一段视频或尾帧控制当前段开场状态。
- 如果实际生成结果与计划不同，以用户接受的实际画面为准。
- 不要让动作参考视频覆盖角色身份；不要让上一段生成漂移后的脸覆盖原始角色图身份。
- 用户上传的生成视频、审核视频或失败样片默认是“分析证据”，不是新的 `@视频` 生成参考。只有用户明确指定它控制开场状态、动作、运镜、节奏或声音时，才把它路由为生成参考。
- 已接受画面可以只验收到某个剪辑点。剪辑点之后的未接受尾部不进入正史；下一段从剪掉后的最后可见状态继续。
- 用户决定“后期修”时，以修后的预期状态作为下一段正史，例如裁切后的构图、移除字幕后的画面或重新混音后的声音状态，而不是继续继承原始瑕疵。

## 参考图图文冲突规则

如果用户同时给参考图和剧情，先静默检查：

- 主体身份是否一致。
- 年龄、性别、服装、职业是否冲突。
- 地点、时代、天气、光线是否冲突。
- 关键道具、车辆、机器、怪物是否冲突。
- 图片动作状态能否承接剧情。
- 图片构图是否会限制剧情必要动作。

重大冲突时必须先问：

```text
我发现参考图与剧情存在冲突：[具体冲突]。
请你选择：
1. 保留参考图内容，我按图修改剧情描述。
2. 保留剧情内容，我提供新的生图提示词来重做参考图。
```

轻微冲突时可以继续，但要写清只借用有用锚点。

## 音频节奏锁定

当用户上传音频时，必须判断它是哪一种：

- `最终BGM + 节拍网格`：音乐作为最终声音，同时控制节奏。
- `无声节奏参考`：只控制动作、切点和镜头速度感，不输出为背景音乐。
- `音效参考`：只参考音效质感和冲击点。
- `环境声底`：作为环境声底。
- `声线参考`：作为角色或旁白声线。

提示词必须出现等价语句，默认使用中文可见表达：

```text
音频节奏锁定：@音频1 只是无声节奏参考，只控制节拍网格、动作重音、冲击点、切点和镜头速度感；不要把该参考音频输出为背景音乐。最终可听声音是现场机器轰鸣、干燥稻秆摩擦和短促静默。
```

如果音频是最终音乐：

```text
音频节奏锁定：@音频1 是最终BGM和节奏参考，只控制音乐节奏和切点重音，不控制角色身份、服装、地点、台词或镜头。
```

## 声线锁定

参考音频或视频人声作为配音时：

```text
声线锁定：@音频1 是 [旁白/角色名] 的声线参考。跨段保持音色、年龄感、口音、语速、情绪表达和麦克风质感一致。声线只控制声音，不控制脸、服装、地点或台词内容；内心独白/旁白不让画面角色对口型，除非角色正在画面中说话。
```

## 维度锁定写法

角色图：

```text
@图片1 命名为老人A，控制老人脸型、灰白短发、褪色蓝色外套、佝偻姿态。
```

场景图：

```text
@图片2 命名为大棚窄道场景，控制大棚宽度、金属管网格、塑料薄膜质感和泥泞窄道。
```

视频动作参考：

```text
@视频1 命名为动作参考A，控制 lateral tracking speed、踉跄后恢复节奏和前景遮挡切点。
```

高风险视频参考：

```text
@视频1 命名为动作参考A，只控制 lateral tracking speed 和动作节奏；不控制人物身份、服装、地点、色彩或声音。
```

## 防止参考素材漂移

- 参考图里没有的物体如果必须出现，要写“new object enters this shot”。
- 参考音频是 silent guide 时，后文不能写“使用该参考音乐作为 BGM”。
- 同一角色多段输出时，每段都重复角色锚点，不依赖 GPT 记忆。
- 多张参考图同时存在时，为每张分配唯一职责，避免互相覆盖。
- 锁定名建立后，时间轴用锁定名承接，不把每个 time beat 写成 @素材清单。
- 参考素材默认不显性列出所有排除维度，但内部必须保持维度隔离；一旦发现可能带偏生成，就把具体“不控制什么”写进提示词。
